# همپا
## توضیحات: 
ااین اپلیکیشن در زمینه خدمات رسانی در ایام اربعین حسینی(ع)
فعالیت دارد.
حوزه های فعالیت این اپلیکیشن در زمینه خدمات رسانی، رزرو
پارکینگ، رزرو هتل، موقعیت موکب ها و...حوزه های فعالیت این
اپلیکیشنمیباشد
