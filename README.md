# همپا
## توضیحات: 
این اپلیکیشن در زمینه خدمات رسانی در ایام اربعین حسینی(ع)
فعالیت دارد.

پارکینگ، رزرو هتل، موقعیت موکب ها و...حوزه های فعالیت این
اپلیکیشن میباشد.
